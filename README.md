- **API**: TMDB API
- **Version Control**: Git
- **Hosting**: Netlify, Github Pages, Vercel, CloudFlare

## ✨ Features :
- Browse movies by genre
- Search for specific movies
- View movie details, including plot summary, cast, and ratings
